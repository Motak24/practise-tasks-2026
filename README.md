# Выполнение индивидуального задания

**Индивидуальное задание:** разработка сервиса для автоматизации процесса бронирования в сети парикмахерских.

Система включает следующие возможности:

* регистрация пользователей;
* создание заявок на бронирование;
* выбор мастера;
* выбор даты и времени;
* хранение заявок;
* изменение статусов заявок администратором.

Исходный код проекта размещён на GitHub с фиксацией нескольких осмысленных коммитов.
